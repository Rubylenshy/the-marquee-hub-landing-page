# Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Tablet: 780px
- Desktop: 1440px

## Colors

### Primary

- Desaturated green: `hsl(193, 34%, 30%)`

### Neutral

- white: `hsl(0, 33%, 98%)`
- dark-gray: `hsl(192, 5%, 63%)`

## Typography

### Fonts

- Family: [Rubik](https://fonts.google.com/specimen/Rubik)
- Weights: 400, 500, 700
- Body Font size: 16px
